1
Forward Pass — The Attempt
Model tries to predict the next word
Input: "The cat sat on the"
         ↓ tokenize → embed → attention → FFN
Output probabilities:
"floor"
34% ← model picks this
"dog"
8%
"roof"
6%
"mat" ✓
12% — should be highest!
HOW WRONG?
2
Calculate the Loss
Cross Entropy Loss — a single number for "how wrong"
Correct answer: "mat" → should be 100%
Model predicted: "mat" → only 12%

Loss = distance between prediction and truth
Loss value:
HIGH ↑
WHO'S RESPONSIBLE?
3
Backpropagation — Blame Assignment
Work backwards through every layer
↑ Output Layer gradient: −0.42
↑ Transformer Block 12 gradient: −0.31
↑ Transformer Block 6 gradient: −0.18
↑ Embedding Layer gradient: −0.09
// PyTorch does all of this in one line:
loss.backward() ← calculates all gradients
NOW FIX IT
4
Gradient Descent — Tiny Nudge
Every weight shifts slightly in the right direction
Before: [0.12, −0.45, 0.33, 0.67, ...]
       ↓ nudge by learning rate (0.0001)
After:  [0.13, −0.43, 0.35, 0.68, ...]
optimizer.step() ← apply the nudge
optimizer.zero_grad() ← reset for next round
REPEAT BILLIONS OF TIMES
5
After Millions of Iterations
Loss drops → model gets smarter
Early training:  "The cat sat on the ___" → "dog" (random)
Mid training:   "The cat sat on the ___" → "floor" (warmer)
Late training:  "The cat sat on the ___" → "mat" ✓ nailed it
